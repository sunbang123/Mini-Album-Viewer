### WinForms (C#): 빠르고 직관적인 레이아웃 구축

WinForms는 컨트롤들을 드래그 앤 드롭으로 빠르게 배치할 수 있어, 여러 장의 이미지를 관리하는 대시보드를 만들기에 좋습니다.

- **프로젝트 아이디어: 미니 앨범 뷰어 (썸네일 & 변환기)**
- **구현 목표:**
    - 특정 폴더에 있는 여러 사진을 한 번에 불러와 좌측에 작은 썸네일(리스트)로 보여줍니다.
    - 썸네일을 클릭하면 우측 메인 `PictureBox`에 원본 사이즈로 출력합니다.
    - 버튼을 통해 이미지를 90도 회전하거나, 좌우 반전(Flip) 시킵니다.
- **비전(Vision) 학습 포인트:** 실제 비전 검사 장비에서 여러 개의 **검사 결과 로그나 캡처된 불량 이미지 리스트를 UI에 뿌려주는 방식**을 미리 연습할 수 있습니다.
